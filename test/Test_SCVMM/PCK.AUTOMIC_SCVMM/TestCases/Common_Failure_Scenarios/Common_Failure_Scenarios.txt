    ...    -Precondition: To run this test suite SCVMM credentials should be known to user.

    ...    -${Port}: This field specifies port which will be used for connection.
    ...    -${VM_Username}: This field specifies username which will be used for this connection.
    ...    -${VM_Password}: This field specifies password for the given username.
    ...    -${VM_Name}: This field specifies name of the virtual machine you want to get
    ...    -${VM_Host}: This field specifies the host from which you want to get VM.
